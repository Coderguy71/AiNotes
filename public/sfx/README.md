# Sound Effects

## xp-chime.mp3

This should be a light, pleasant sound effect (1-2 seconds) that plays when XP is earned.
Suggested characteristics:
- Positive, uplifting tone
- Short duration (0.5-2 seconds)
- Not too loud or jarring
- Similar to notification chimes in productivity apps

For development, you can use a free sound from:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

Place the audio file as: `public/sfx/xp-chime.mp3`
